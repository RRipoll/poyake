Overview

Poyake is a Low-Code Application Development Framework and a set of tools.

- Create your data model using a text DSL (Domain Specific Language).

- Reverse engineer your existing database.

- Generate ORM classes and database scripts.

- Generate UML ERD (Entity relationship diagrams).

- Create your front end using either REST or ZK

- Eclipse Plugin to edit DSL, Code generator, ORM Engine, REST interface, ZK UI and an automatic testing tool (TestRunner).

Features

Change resilient

Full round trip engineering makes it easy to change your code in sync with your documentation and data structures with full control

No vendor lock-in

We are open source and use the standard java technology stack

Source control friendly

Designed to work with big teams and source control systems. No binary model files. Easy to merge

Up to date documentation

Documentation is kept up to date as it is generated from the data structures.

No Database impedance.

Work with views without losing metadata information. Views behave as entities. Insert or update into views simply.

Testing Tools.

A testing tool is available with the framework with test definition files designed to work well with source control systems.
